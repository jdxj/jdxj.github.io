  • 标准输出的文件描述符为1

POSIX规范的正则表达式#

1

GNU规范#

  • \b（边界字符，匹配单词的开始或结尾）
  • \B（与\b为反义词，\Bthe\B不会匹配单词the，仅会匹配the在中间的单词，如atheist）
  • \w（等同于[_[:alnum:]]）
  • \W（等同于[^_[:alnum:]]）
  • \d表示任意数字
  • \D表示任意非数字
  • \s表示任意空白字符（空格、制表符等）
  • \S表示任意非空白字符
1
2

#匹配i结尾的单词
$ grep "i\b"  /tmp/passwd
