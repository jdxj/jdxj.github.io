与正则表达式有关的问题很少会有什么终极答案。

## 11.6 IP地址

```regexp
(((\d{1,2})|(1\d{2})|(2[0-4]\d)|(25[0-5]))\.){3}
➥((\d{1,2})|(1\d{2})|(2[0-4]\d)|(25[0-5]))
```

## 11.7 URL

```regexp
https?:\/\/[-\w.]+(:\d+)?(\/([\w\/_.]*)?)?
```

## 11.8 完整的URL

```regexp
https?:\/\/(\w*:\w*@)?[-\w.]+(:\d+)?(\/([\w\/_.]*(\?\S+)?)?)?
```

就性能来说，越复杂的模式，执行速度越慢。

## 11.9 电子邮件地址

```regexp
(\w+\.)*\w+@(\w+\.)+[A-Za-z]+
```
